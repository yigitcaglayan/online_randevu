# \# 📅 Online Randevu ve İşletme Yönetim Sistemi

# 

# !\[Uygulama Vitrini](https://via.placeholder.com/1200x400?text=Online+Randevu+Sistemi+Gosterimi)

# 

# Bu proje, işletmelerin dijitalleşmesini sağlayan, \*\*Flask\*\* altyapısıyla geliştirilmiş modern bir randevu yönetim platformudur. İşletme sahiplerine yönetim kolaylığı, müşterilere ise hızlı erişim sunar.

# 

# \---

# 

# \## ✨ Öne Çıkan Özellikler

# 

# \### 🏢 İşletme Paneli

# \* \*\*Sınırsız İşletme:\*\* Tek hesapla birden fazla işletme/şube yönetimi.

# \* \*\*Akıllı Planlama:\*\* Çalışma saatleri ve hizmet açıklamalarını özelleştirme.

# \* \*\*Merkezi Randevu Takibi:\*\* Tüm randevuları anlık olarak görebilme.

# 

# \### 👥 Müşteri Paneli

# \* \*\*İşletme Keşfi:\*\* Şehirdeki tüm kayıtlı noktaları listeleyebilme.

# \* \*\*Hızlı Rezervasyon:\*\* Uygun saatleri seçerek saniyeler içinde randevu alma.

# \* \*\*Takip Sistemi:\*\* Mevcut randevuların durumunu kişisel panelden izleme.

# 

# \---

# 

# \## 🛠️ Teknik Detaylar

# 

# \* \*\*Backend:\*\* Python 3.x, Flask Framework

# \* \*\*Veritabanı:\*\* SQLite \& SQLAlchemy (ORM)

# \* \*\*Arayüz:\*\* Jinja2, Modern CSS3 (Dark Theme \& Glassmorphism)

# \* \*\*Güvenlik:\*\* Flask-Login (Oturum Yönetimi) \& Werkzeug (Password Hashing)
